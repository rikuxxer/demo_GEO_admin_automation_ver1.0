## 1-1. GEO Targeting とは

最小半径50mで複数の地点をカスタマイズ性高くターゲティングできるエリアマーケティング手法。

### 他社との差別化ポイント

- Google や Facebook のジオターゲティングは半径1km単位だが、UNIVERSE GEO は**半径1m**から指定可能
- 来店計測が容易にできる

### GEO プロダクト一覧

| プロダクト名 | 概要 |
|------------|------|
| **UNIVERSE GEO** | UNIVERSE Ads のジオターゲティング |
| **MicroAd Geo Targeting** | UNIVERSE ではなく SNS で配信する場合に使用 |

> **注意（パチンコホール訴求）**：UNIVERSEのunerryデータはパチンコ案件を利用できません。BWへの依頼をお願いします

## 1-2. SIM 作成・EAGLE について

### EAGLE とは

GEO Targeting の **id数を確認できるツール**。Ursula やチャットを介さずに id 数を確認でき、代理店との提案時にその場で id 数の算出が可能。

- **URL**: https://market.profilepassport.jp/segment
- **ID**: gm@microad.co.jp
- **PASS**: 9TqCmqG8

### 居住者・勤務者の id 数 計算例

```
EAGLE 100万id → 100万 × 3倍 = 300万UB → 300万 × 1/10 = 30万UB
```

### 管理画面に表示されない条件

来店の頻度指定・期間指定などは管理画面に表示されないため、チャットにて事業部に相談するか、係数表を確認すること。

## 1-3. 提案にあたっての確認事項

### 提案前チェック

1. 提案不可の案件でないかを確認する
2. レギュレーション・キャンペーンシートの作成方法を確認する

### 禁止セグメント（地点）

人権の侵害や差別を助長するセグメント（地点）は**禁止**。

- 宗教施設や特定の思想を有する学校
- 人の生死に関する行事や施設（葬儀場など）
- 犯罪経歴を特定できるもの
- 中毒に関わるもの
- 妊娠/出産/中絶を特定できるもの
- 病歴を特定できる病院・専門クリニック（メンタルクリニック/産婦人科など）

> **LP について**: LP の判断は MA 判断となる。

### 来店計測のレギュレーション

| 項目 | ルール |
|------|--------|
| **最低出稿金額** | 30万円〜 |
| **来店計測地点の追加** | 10万円ごとに追加可能 |
| **来店計測期間** | 配信期間 = 来店計測期間（それ以外はイレギュラー → ビジ開に相談） |
| **計測期間の変更** | **2営業日前**までに事業部に連絡 |

## 位置情報広告の基礎知識

- **検知**: スマートフォンの位置情報がPOIから指定半径内に入った状態を指す
- **TG地点（ターゲティング地点）**: 広告配信のターゲットとなる地点。セグメントの抽出条件が適用される
- **来店計測地点**: 広告効果（来店率）を計測するための地点。配信対象とは別に設定する
- **セグメント**: 配信設定の単位。1案件に複数のセグメントを持てる
- **UU（ユニークユーザー）**: 重複を除いたユニークなデバイス数。配信規模の指標

## データ連携ステータスの意味

| ステータス値 | 表示名 | 説明 |
|------------|--------|------|
| `before_request` | 依頼前 | まだデータ連携依頼が出ていない状態 |
| `requested` | 依頼済 | データ連携依頼が完了し、管理部が連携作業中 |
| `linked` | 連携済 | データ連携が完了し、配信可能な状態 |

---

## 案件ステータスの意味

| ステータス値 | 説明 |
|------------|------|
| `draft` | 案件作成中・未確定 |
| `in_progress` | 配信進行中 |
| `completed` | 配信完了 |
| `cancelled` | キャンセル |

---

## よく使われる用語

- **格納依頼**: 地点データを配信システムに取り込む依頼。完了後はTG地点の編集不可になる
- **抽出期間**: セグメントの対象ユーザーを抽出する期間（例：過去30日間に検知されたユーザー）
- **指定半径**: 地点周辺の検知エリアの半径（メートル単位）
- **配信媒体**: 広告を配信するプラットフォーム（例：アドネットワーク、DSP）
- **ads_account_id**: 配信プラットフォームのアカウントID
