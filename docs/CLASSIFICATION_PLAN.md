# ドキュメント分類計画

## 分類基準

### 開発環境 (dev/)
- ローカル開発環境のセットアップ
- テスト環境の設定
- デバッグ手順
- 開発用の設定変更
- 開発中のトラブルシューティング

### 本番環境 (prod/)
- 本番環境のデプロイ
- 本番環境の設定
- 本番環境のトラブルシューティング
- リリース準備
- 本番環境の運用

### 共通 (shared/)
- 両方の環境に適用される情報
- BigQueryスキーマ定義
- 一般的なトラブルシューティング
- API仕様
- アーキテクチャ
