# 開発環境トラブルシューティング

このディレクトリには、開発環境に関するトラブルシューティングドキュメントが含まれています。

## カテゴリ

### テスト・デバッグ
- `TEST_API_ENDPOINTS.md` - APIエンドポイントのテスト方法
- `TEST_SHEETS_OUTPUT.md` - スプレッドシート出力のテスト
- `TEST_ENVIRONMENT_INFO.md` - テスト環境の情報
- `TEST_ENVIRONMENT_URLS.md` - テスト環境のURL
- `QUICK_TEST.md` - クイックテスト手順
- `SPREADSHEET_EXPORT_TEST.md` - スプレッドシートエクスポートのテスト

### デバッグ
- `DEBUG_PROJECT_CREATION.md` - プロジェクト作成のデバッグ
- `DEBUG_USER_REGISTRATION_ERROR.md` - ユーザー登録エラーのデバッグ
- `CHECK_DEMO_DATA.md` - デモデータの確認

### 環境設定
- `MANUAL_ENV_SETUP.md` - 手動環境設定
- `FIX_ENV_VARIABLES.md` - 環境変数の修正
- `SET_ENV_FOR_FRONTEND_URL.md` - フロントエンドURLの環境変数設定
- `MANUAL_SET_ENV_VARS.md` - 環境変数の手動設定
